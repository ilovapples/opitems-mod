## Build
`git clone` the repo, navigate to it, and run one of the following in the terminal/command prompt:
- On Windows:
  - `gradlew.bat build`
- On Linux/MacOS X:
  - `./gradlew build`
  
The .jar file will be in the build/libs directory.
